# 🛒 Projeto E-commerce - ÂMAGO

## 📌 Descrição

Este projeto consiste no desenvolvimento de um pequeno site de e-commerce utilizando apenas HTML e CSS, conforme solicitado na atividade de Programação Web.

A loja fictícia **ÂMAGO** tem como objetivo apresentar camisetas em estilo moderno e urbano, com foco em organização, navegação entre páginas e estrutura básica de um sistema de compras.

---

## 🎯 Objetivo da Atividade

Criar um site contendo:

* Página inicial com produtos em destaque e uma notícia
* Página com listagem de produtos
* Página simulando um carrinho de compras
* Página de contato com formulário
* Navegação entre todas as páginas
* Estrutura com cabeçalho e rodapé

---

## 🧩 Estrutura do Projeto

O projeto foi dividido em 4 páginas principais:

### 🏠 index.html (Página Inicial)

* Apresenta:

  * Dois produtos em destaque
  * Uma promoção
  * Uma notícia
* Introduz o usuário à loja

---

### 🛍️ produto.html (Listagem de Produtos)

* Exibe diversos produtos disponíveis
* Cada produto contém:

  * Imagem
  * Nome
  * Descrição
  * Preço
  * Botão “Adicionar ao carrinho”

---

### 🛒 carrinho.html (Carrinho)

* Simula produtos adicionados
* Exibe:

  * Nome dos produtos
  * Quantidade
  * Preço
  * Valor total
* Possui botão de finalização de compra

---

### 📞 contato.html (Fale Conosco)

* Formulário com os campos:

  * Nome
  * Email
  * Telefone
  * Assunto
  * Mensagem

---

## 🎨 Tecnologias Utilizadas

* HTML5
* CSS3

> O projeto foi desenvolvido sem uso de JavaScript, conforme exigência da atividade.

---

## 🧱 Estrutura Visual

* Cabeçalho com:

  * Logo da loja
  * Nome ÂMAGO
  * Menu de navegação

* Rodapé com:

  * Nome do desenvolvedor
  * Telefone
  * Email

* Layout com:

  * Produtos organizados em blocos (divs)
  * Centralização dos elementos
  * Destaque visual para promoções

---

## 🔗 Navegação

Todas as páginas estão interligadas por meio de um menu fixo, permitindo ao usuário navegar entre:

* Início
* Produtos
* Carrinho
* Contato

---

## 💡 Funcionalidades

* Simulação de carrinho (sem lógica de programação)
* Navegação entre páginas HTML
* Exibição de produtos com informações completas
* Formulário de contato funcional (estrutura HTML)
